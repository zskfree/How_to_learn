# 文献检索工作流

推荐把文献检索拆成“发现、筛选、获取、阅读、整理”五步。

## 1. 发现

优先使用：

- `科研平台.google_scholar`
- `科研平台.semantic_scholar`
- `科研平台.openalex`

用主题词、同义词、作者名和关键论文标题组合搜索。

## 2. 筛选

关注：

- 发表年份。
- 期刊或会议质量。
- 引用数量和引用来源。
- 摘要是否匹配研究问题。
- 是否有开放全文或机构授权全文。

## 3. 获取

优先顺序：

1. 开放获取版本。
2. 作者主页或机构仓储版本。
3. 学校、机构或图书馆订阅。
4. 合法文献传递或馆际互借。

## 4. 阅读

可以用 `AI平台.chatgpt`、`AI平台.claude`、`AI平台.gemini` 辅助提取论文结构、方法、结论和局限，但关键结论必须回到原文核对。

## 5. 整理

建议使用 `工具.zotero` 管理文献，并记录：

- 研究问题。
- 核心结论。
- 方法和数据。
- 与自己课题的关系。
- 待验证问题。

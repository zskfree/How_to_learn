# 连通性检查清单

安装或调整 Clash Verge 后，按下面顺序检查。

## 1. 软件状态

- Clash Verge 已启动。
- 配置文件已导入。
- 已选择可用节点。
- 模式为规则模式或当前需要的模式。
- 系统代理或 TUN 模式状态符合预期。

## 2. 出口检查

引用 ID：

- `工具.ipinfo`

检查当前出口 IP、地区和网络状态是否符合预期。

## 3. 核心平台检查

优先检查：

- `科研平台.google_scholar`
- `AI平台.chatgpt`
- `AI平台.claude`
- `AI平台.gemini`

如果这几个平台都能正常访问，说明科研访问的基础链路大概率可用。

## 4. 问题定位

- 只有某个平台打不开：优先查平台状态、账号状态、节点地区和浏览器缓存。
- 大部分国外网站打不开：优先查 Clash Verge 是否启动、节点是否可用、系统代理是否生效。
- 国内网站异常：检查是否误用了全局模式，或规则配置过于激进。

# 连通性验证命令

本页提供基础验证思路。命令只用于检查连通性，不修改系统配置。

## PowerShell 基础检查

```powershell
Test-NetConnection scholar.google.com -Port 443
Test-NetConnection chatgpt.com -Port 443
Test-NetConnection claude.ai -Port 443
Test-NetConnection gemini.google.com -Port 443
```

## 浏览器检查

优先用浏览器实际打开：

- `工具.ipinfo`
- `科研平台.google_scholar`
- `AI平台.chatgpt`
- `AI平台.claude`
- `AI平台.gemini`

## 判断结果

- TCP 成功但网页打不开：可能是浏览器、账号、平台策略或规则问题。
- TCP 失败且多个平台失败：优先检查 Clash Verge 状态和节点。
- 只有一个平台失败：优先检查该平台状态、账号权限和节点地区。

# 常见网络错误

## 页面无法打开

可能原因：

- Clash Verge 未启动。
- 节点不可用。
- 当前模式不符合预期。
- 浏览器缓存或扩展影响访问。
- 平台本身暂时不可用。

处理顺序：

1. 检查 Clash Verge 是否运行。
2. 切换节点。
3. 检查 `工具.ipinfo`。
4. 检查 `科研平台.google_scholar`。
5. 检查 `AI平台.chatgpt`、`AI平台.claude`、`AI平台.gemini`。

## 国内网站变慢

可能是误用了全局模式，或规则配置把国内流量也转发出去了。优先切回规则模式。

## 只有某个平台异常

不要立即改系统设置。先检查平台登录状态、账号权限、浏览器缓存、节点地区和平台服务状态。

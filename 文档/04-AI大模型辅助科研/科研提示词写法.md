# 科研提示词写法

好的提示词要告诉 AI：任务、材料、输出格式和限制。

## 基本模板

```text
你是科研助理。请基于我提供的材料完成任务。

任务：
- 总结这篇论文的研究问题、方法、数据、结论和局限。

要求：
- 不要编造材料中没有的信息。
- 不确定的地方标注“原文未说明”。
- 输出为 Markdown 表格。
```

## 论文阅读模板

```text
请阅读下面论文内容，输出：
1. 研究问题
2. 核心假设
3. 数据来源
4. 方法设计
5. 主要发现
6. 局限性
7. 我需要回到原文核对的关键句子
```

## 文献对比模板

```text
请把下面多篇论文按以下字段对比：
- 作者和年份
- 研究对象
- 方法
- 数据
- 主要结论
- 与我的课题的关系
- 可信度或局限
```

## 平台建议

- 综合任务：`AI平台.chatgpt`
- 长文档：`AI平台.claude`
- Google 生态和多模态：`AI平台.gemini`
- 带来源搜索：`AI平台.perplexity`、`AI平台.consensus`、`AI平台.elicit`

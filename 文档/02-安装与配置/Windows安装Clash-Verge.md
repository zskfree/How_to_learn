# Windows 安装 Clash Verge

本项目已将安装包放在：

- `安装包/Clash.Verge_2.5.1_x64-setup.exe`

统一引用 ID：

- `工具.clash_verge`

## 安装步骤

1. 双击安装包。
2. 按安装向导完成安装。
3. 从开始菜单或桌面快捷方式启动 Clash Verge。
4. 如系统提示防火墙或网络权限，根据可信来源和实际需求确认。

## 安装后检查

- 能正常打开 Clash Verge 主界面。
- 能看到配置、代理、规则、日志等页面。
- 后续可以导入配置文件或订阅。

## 注意事项

- 不要把订阅链接写入本文档。
- 不要在不理解影响的情况下修改系统环境变量。
- 不要默认给 Git、npm、pip、uv 写入全局代理。
- 安装完成不代表网络已可用，必须继续做连通性检查。
